# كيفية بناء ملف APK (تطبيق أندرويد)

تم تجهيز المشروع ليعمل مع Capacitor. اتبع الخطوات التالية لاستخراج ملف APK.

## المتطلبات المسبقة
1. تأكد من تثبيت **Node.js** على جهازك.
2. تأكد من تثبيت **Android Studio** (ضروري لبناء ملف الـ APK النهائي).

## الخطوات

### 1. تثبيت المكتبات
قم بتشغيل الأمر التالي في التيرمنال (Terminal) لتثبيت مكتبات Capacitor المطلوبة:

```bash
npm run android:init
```

### 2. بناء المشروع وفتح Android Studio
قم بتشغيل الأمر التالي. سيقوم هذا الأمر ببناء نسخة الويب، ثم مزامنتها مع مجلد الأندرويد، وفتح Android Studio تلقائياً:

```bash
npm run android:build
```

### 3. استخراج ملف APK
بمجرد أن يفتح **Android Studio**:
1. انتظر حتى تنتهي عملية الفهرسة (Indexing) والـ Gradle Sync في الأسفل.
2. اذهب إلى القائمة العلوية واختر: **Build > Build Bundle(s) / APK(s) > Build APK(s)**.
3. عند الانتهاء، ستظهر رسالة في الأسفل. اضغط على **locate** للذهاب لمجلد ملف الـ APK.
4. يمكنك الآن نقل هذا الملف لهاتفك وتثبيته.

---

### ملاحظات هامة
* **مفتاح API**: تأكد من أن مفتاح `API_KEY` الخاص بـ Gemini مضبوط بشكل صحيح في ملفات البيئة (`.env`) الخاصة بك قبل البناء، حيث سيتم تضمينه داخل التطبيق.
* **الأيقونات**: لتغيير أيقونة التطبيق، ستحتاج لاستبدال الصور في مجلد `android/app/src/main/res/`.
